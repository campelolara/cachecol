# Cachecol – Uma Comunidade Para Leitores
Aplicação mobile desenvolvida com o objetivo de criar um espaço digital acolhedor e interativo para leitores. Desta forma, funcionando como uma rede social voltada à literatura, incentivando a troca de ideias, recomendações e experiências entre leitores.

## Funcionalidades
- Perfis e seguidores: Criação de perfis e possibilidade de seguir outros usuários.
- Publicação de conteúdo: Resenhas, comentários e avaliações.
- Estantes virtuais personalizadas: Organize leituras como lidos, lendo, favoritos etc.
- Interação social: Feed ativo para incentivar debates, recomendações e compartilhamento.
- Proposta

## Tecnologias utilizadas
- FlutterFlow: Desenvolvimento front-end e ações de CRUD.
- Firebase: Banco de dados e autenticação de usuários.

## Status do Projeto
O Cachecol foi inicialmente desenvolvido utilizando ferramentas Low-Code, permitindo rápida validação da ideia e foco na experiência do usuário.

Atualmente, o projeto encontra-se em fase de planejamento para uma nova versão, com o objetivo de:
- Reescrever a aplicação utilizando linguagens e frameworks nativos ou multiplataforma focados em desenvolvimento mobile (e possibilitando expansão futura para desktop e web).
- Aprimorar arquitetura e escalabilidade.
- Explorar novas funcionalidades e melhorias de desempenho.

Esta nova etapa visa transformar o protótipo em uma solução mais robusta, moderna e completa para a comunidade leitora.
